# NOTICE

## What this repository is
A **sanitized public demo package** for the Einstein Brain concept — a governed company-memory / trust layer for AI agents. It exists to explain the architecture, the trust loop, a sample Skill, a runnable demo harness, and the honest limitations.

## What this repository is NOT
- It is **not** the product. The working system runs on a **private corpus** that is not included here.
- It contains **no** private knowledge nodes, **no** customer/company data, **no** secrets/keys, **no** local paths, and **no** vendor-internal references.
- All examples are **fictional** (a made-up company, "Acme Cloud" / "DemoCo").

## Metrics
Any metrics quoted (e.g. retrieval hit rate, abstention rate) are from the maintainer's **private eval set** and are re-runnable on the private system — **not** reproducible from this package alone.

## Status of claims
This is an early-stage demo. The **read** layer (grounded retrieval + abstention + provenance) is real; the governed-**write** loop is in pilot; connectors and executable-skill compilation are roadmap. See `LIMITATIONS.md`.
